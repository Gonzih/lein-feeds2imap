Feed 2 IMAP

Read your RSS/Atom feeds using just email client.
download .ZIPdownload .TGZ

Installation

  • Install leiningen
  • Add [lein-feeds2imap "0.1.6"] to ~/.lein/profiles.clj file.
  • Minimalistic ~/.lein/profiles.cljs should look like that:
{:user {:plugins [[lein-feeds2imap "0.1.6"]]}}

Usage

# This should bootstrap configuration directory `~/.config/feeds2imap.cljs/` and most of configuration files.
lein feeds2imap add Clojure 'http://planet.clojure.in/atom.xml'

Configure your imap account (take a look at [example file]https://github.com/Gonzih/feeds2imap.clj/blob/master/imap.clj.example))

Run it

lein feeds2imap pull
#or
lein feeds2imap auto

For more information check out help lein feeds2imap --help.
