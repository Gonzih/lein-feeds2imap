Feeds 2 IMAP

Read your RSS/Atom feeds using just email client.
download .ZIPdownload .TGZ

Installation

  • Install leiningen
  • Add [lein-feeds2imap "0.1.8"] to ~/.lein/profiles.clj file.
  • Minimalistic ~/.lein/profiles.clj should look like that:
{:user {:plugins [[lein-feeds2imap "0.1.8"]]}}

Usage

  • Add url to folder:
    • # This should bootstrap configuration directory
`~/.config/feeds2imap.clj/`
and most of configuration files.
lein feeds2imap add Clojure 'http://planet.clojure.in/atom.xml'
  • Configure your imap account (take a look at example ~/.config/feeds2imap.clj/imap.clj)
  • Encrypt imap credentials using lein feeds2imap imap encrypt
  • Run it:
    • lein feeds2imap pull
#or
lein feeds2imap auto
  • For more information check out help lein feeds2imap --help.
